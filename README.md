# FMCG-Revenue-Portfolio-Strategy-Analysis
FMCG multi-channel sales dashboard built in Excel 2019 analyzing performance across Channel, Brand, and Category. Features dynamic filtering, KPI tracking (Revenue, ASP, Promo %, Stock Out), and portfolio-level business insights with strategic recommendations. Demonstrates commercial analysis and data modeling skills using Excel only.
